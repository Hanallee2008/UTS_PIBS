1. Pengembangan Web

  • JavaScript: Menguasai JavaScript untuk membuat interaktivitas pada halaman web. Saya berpengalaman dalam DOM manipulation, event handling, serta AJAX untuk komunikasi asinkron dengan server.
  • Front-End Frameworks: Familiar dengan framework seperti Bootstrap dan Tailwind untuk mempercepat pengembangan antarmuka dengan desain yang modern.

2. Pemrograman Back-End

  • Python: Menggunakan Python untuk pengembangan aplikasi berbasis web dan otomasi tugas-tugas tertentu. Saya juga berpengalaman menggunakan framework Django dan Flask untuk pengembangan back-end.

  • Manajemen Keuangan: Memahami dasar-dasar manajemen keuangan dan budgeting, serta berpengalaman dalam menyusun laporan keuangan sederhana untuk organisasi kampus.
  • Version Control (Git): Mampu menggunakan Git dan GitHub untuk kolaborasi proyek, version control, serta deployment aplikasi ke server atau platform cloud.

  • Komunikasi: Mampu berkomunikasi dengan jelas, baik secara tertulis maupun lisan. Saya terbiasa mempresentasikan ide-ide secara profesional di depan tim dan klien.
